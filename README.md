# IWANBOTWA
Ini bot wa base Stiker saja untuk fitur silahkan tambah sendiri

## Install via Termux
* pkg update && pkg upgrade
* pkg install nodejs
* pkg install python
* pkg install ffmpeg
* pkg install git

## Lakukan clone repository

* git clone https://github.com/wansky05/iwanbotwa
* cd iwanbotwa
* npm install
* node index.js
